# Giftbloom— Mobile App UX/UI Case Study

GiftBloom is a mobile app prototype designed to simplify gift-giving through personalized recommendations, wishlists, and event reminders. The project focuses on reducing the stress of choosing meaningful gifts by applying user-centered design principles and intuitive mobile UI patterns.
 
**Figma Prototype:**  
https://www.figma.com/proto/eYsNSwtN95RIgR8rAqFfl8/GiftBloom?node-id=0-1&t=KzptECSipmdshsTA-1

---

## Problem

Many people struggle with gift-giving due to forgotten dates, uncertainty about preferences, or lack of inspiration—especially in group or workplace settings.

GiftBloom addresses this by providing:
- Guided gift discovery
- Shareable wishlists
- Built-in event reminders

---

## Key Features

- Interest-based gift quiz  
- Shareable wishlist  
- Calendar for birthdays & holidays  
- User account management  
- Contact-based planning
